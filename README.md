# midairywala.com

Personal website of Mohammed I. Dairywala, MD — integrated vascular surgery resident at Henry Ford Health Providence Hospital & Michigan State University College of Human Medicine.

A single static page (no build step) covering an about section, publications, operative videos, and training. Hosted on Vercel.

## Files

- `index.html` — the site (HTML, CSS, and JS in one file)
- `favicon.svg`, `favicon.png`, `apple-touch-icon.png` — icons
- `og-image.png` — social share preview image
- `robots.txt`, `sitemap.xml` — search-engine files
- `vercel.json` — hosting config (clean URLs + security headers)
- `DEPLOY.md` — deployment and DNS instructions

## Editing

Edit `index.html` and redeploy (`vercel --prod`, or push to GitHub if connected to Vercel).
